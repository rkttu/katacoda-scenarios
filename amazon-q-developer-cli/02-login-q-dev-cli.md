# 2. Q Developer CLI 로그인

이 단계에서는 Q CLI를 사용하여 Amazon Q 서비스에 로그인합니다.

## 1. Q CLI 로그인 명령 실행

다음 명령어를 실행하여 Q 서비스에 로그인합니다:

```bash
q login
```{{exec}}

명령어를 실행하면 웹 브라우저가 열리고, AWS 계정으로 인증할 수 있는 페이지로 리다이렉트됩니다.

## 2. 로그인 확인

아래 명령어로 로그인 상태를 확인할 수 있습니다:

```bash
q whoami
```{{exec}}

사용자 정보가 출력되면 로그인에 성공한 것입니다.
